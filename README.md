# Tugas Dasar Function - Bahasa Dart

Tugas ini berisi contoh penggunaan 4 jenis fungsi pada 3 topik berbeda.

### Penjelasan Jenis Fungsi:
1. **Void Tanpa Parameter**: Fungsi yang langsung melakukan aksi (seperti print) tanpa butuh data tambahan.
2. **Return Tanpa Parameter**: Fungsi yang memberikan nilai (hasil) tanpa butuh data tambahan.
3. **Void Dengan Parameter**: Fungsi yang butuh data tambahan (input) untuk diproses.
4. **Return Dengan Parameter**: Fungsi paling lengkap yang butuh input dan memberikan nilai hasil prosesnya.

### Cara Menjalankan:
Gunakan perintah `dart main.dart` pada terminal.
